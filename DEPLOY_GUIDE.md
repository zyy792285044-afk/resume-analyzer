# 简历战斗力分析 Agent - 部署指南

## 🚀 发布到 Coze 平台

### 方式一：网页端快速发布（推荐新手）

#### 第一步：登录 Coze 平台

访问：https://www.coze.cn（国内）或 https://www.coze.com（国际版）

#### 第二步：创建 Bot

1. 点击右上角 **"创建 Bot"**
2. 填写信息：
   - Bot名称：`简历战斗力分析`
   - 描述：`专业简历分析工具，快速识别简历问题，提供优化建议`
   - 图标：上传一个简历相关的图标

#### 第三步：配置 Bot

**1. 选择模型**
```
模型：doubao-seed-2-0-pro
Temperature: 0.3
Top P: 0.9
Max Tokens: 16000
开启 Thinking 模式
```

**2. 填写 System Prompt**

复制以下内容到 System Prompt：

```markdown
# 角色定义
你是一名极其专业的简历战斗力分析专家，拥有10年大厂招聘经验。你的任务是帮助求职者快速识别简历的致命问题，并给出具体可执行的改进建议。

...（复制 config/agent_llm_config.json 中的 sp 字段内容）
```

**3. 配置输入参数**

创建三个输入变量：

| 变量名 | 类型 | 描述 | 必填 |
|--------|------|------|------|
| resume_input | 文本（多行） | 请粘贴简历文本或提供PDF链接 | 是 |
| target_role | 文本 | 目标岗位 | 是 |
| candidate_level | 下拉选择 | 候选人级别（初级/中级/高级） | 否 |

**4. 添加插件（可选）**

如果你的简历是 PDF 格式：
- 添加 "文件处理" 插件
- 或添加 "URL 解析" 插件

#### 第四步：测试 Bot

输入测试数据：
```
简历内容：
张三，5年产品经验
工作经历：2020-2023 某科技公司 产品经理
目标岗位：AI产品经理
候选人级别：中级
```

检查输出是否符合预期。

#### 第五步：发布 Bot

1. 点击右上角 **"发布"**
2. 选择发布渠道：
   - ✅ **网页链接**（最重要，用户可以直接访问）
   - ✅ **API 接口**（可选，供开发者调用）
   - 其他：飞书、微信等（可选）

3. 设置访问权限：
   - 公开：任何人都可以使用
   - 私密：仅自己或指定人员使用

4. 点击"确认发布"

#### 第六步：获取链接

发布成功后，你会得到：

**网页链接：**
```
https://www.coze.cn/s/xxxxx
```

用户打开这个链接就能直接使用你的 Agent！

**API 调用信息：**
```
API Endpoint: https://api.coze.cn/v1/chat
Bot ID: xxxxx
```

### 方式二：使用代码项目部署（开发者模式）

如果你想用自己的服务器部署，可以这样做：

#### 1. 准备服务器

**推荐配置：**
- CPU: 2核
- 内存: 4GB
- 带宽: 5Mbps
- 价格: 约 100-200元/月

**服务商：**
- 阿里云 ECS
- 腾讯云 CVM
- 华为云 ECS

#### 2. 部署步骤

```bash
# 1. 克隆代码
git clone <your-repo-url>
cd resume-analyzer

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
export COZE_WORKLOAD_IDENTITY_API_KEY="your-api-key"
export COZE_INTEGRATION_MODEL_BASE_URL="https://api.coze.cn"

# 4. 启动服务
python src/main.py -m http -p 5000

# 5. 访问服务
# API地址：http://your-server-ip:5000
```

#### 3. 配置域名（可选）

如果你有自己的域名：

1. 购买域名（阿里云、腾讯云等）
2. 备案（国内服务器必须，约2-3周）
3. 配置 DNS 解析到服务器 IP
4. 配置 HTTPS 证书（Let's Encrypt 免费）

### 方式三：使用 Serverless 部署（最省钱）

#### 使用阿里云函数计算 FC

```bash
# 1. 安装 Fun 工具
npm install -g @alicloud/fun

# 2. 创建函数
fun init -n resume-analyzer

# 3. 部署函数
fun deploy

# 4. 获得 HTTP 触发器 URL
# https://xxxxx.cn-hangzhou.fc.aliyuncs.com/xxxxx
```

**优势：**
- 按调用次数计费（很便宜）
- 不需要维护服务器
- 自动扩缩容
