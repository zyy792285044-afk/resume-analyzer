# 📱 Coze平台发布指南

## 🎯 发布后的效果

发布成功后，您将获得：

| 发布渠道 | 获得内容 | 使用场景 |
|---------|---------|---------|
| **Bot商店** | 公网访问链接 | 直接分享给用户 |
| **微信小程序** | 小程序码/链接 | 嵌入公众号菜单/文章 |
| **API服务** | API接口地址 | 集成到自有平台 |

---

## 📝 发布步骤（详细图文版）

### 第一步：登录Coze平台

1. 访问：**https://www.coze.cn**
2. 使用您的账号登录（微信/手机号）

---

### 第二步：找到Agent

1. 进入工作空间
2. 在「我的Bot」列表中找到：**简历战斗力分析**
3. 点击进入Bot编辑页面

---

### 第三步：检查Bot配置

在发布前，请确认以下配置：

**✅ Bot名称**
```
简历战斗力分析
```

**✅ Bot简介**（重要！会显示在分享卡片上）
```
专业简历分析助手，5分钟深度诊断你的简历问题，提供战斗力评分、四维拆解、优化建议和PDF报告下载。
```

**✅ Bot头像**
- 建议上传专业的图标
- 或使用平台默认头像

**✅ 开场白**（用户首次打开时显示）
```
👋 你好！我是简历战斗力分析助手

请直接粘贴你的简历内容，我将为你进行专业分析：
• 📊 战斗力总评（0-100分）
• 🎯 四维能力拆解
• ⚠️ 三大致命问题诊断
• ✨ 改前VS改后优化建议
• 📄 PDF报告下载

现在就开始吧！粘贴简历内容并发送即可。
```

---

### 第四步：发布Bot

1. 点击右上角**「发布」**按钮

2. 选择发布渠道：

   **方案A：发布到Bot商店（推荐）**
   - ✅ 勾选「Bot商店」
   - 选择分类：「效率工具」
   - 标签：「简历」「求职」「面试」
   - 点击「发布」

   **方案B：发布为微信小程序**
   - ✅ 勾选「微信小程序」
   - 授权微信公众号（可选）
   - 点击「发布」

   **方案C：发布为API服务**
   - ✅ 勾选「API服务」
   - 获取API Key和接口地址
   - 点击「发布」

---

### 第五步：获取访问链接

发布成功后：

**方式1：Bot商店链接**
```
https://www.coze.cn/store/bot/您的BotID
```
- 直接复制分享给用户
- 用户点击即可对话使用

**方式2：小程序码**
- 在Bot详情页下载小程序码
- 插入公众号文章或小红书笔记

**方式3：API接口**
- 获取API Key和接口地址
- 用于集成到自有平台

## ⚠️ 常见问题

### Q: 发布后链接打不开？
A: 等待1-2分钟，平台需要审核和部署

### Q: 小程序码无法识别？
A: 确保已授权微信公众号，或使用Bot商店链接

### Q: 用户反馈响应慢？
A: 首次调用可能有冷启动，后续会更快

### Q: 如何查看使用数据？
A: 在Coze后台「数据分析」查看调用次数、用户反馈等

## 📊 推荐配置（可选优化）

如果需要优化Bot性能，可以调整：

**config/agent_llm_config.json**
```json
{
  "config": {
    "temperature": 0.7,  // 降低到0.5会更严谨
    "max_completion_tokens": 8000  // 控制输出长度
  }
}
```

修改后需要重新发布Bot。
