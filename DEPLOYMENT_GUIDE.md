## 🚀 方式一：本地启动（推荐用于测试）

### 步骤

1. **启动本地服务器**
```bash
cd /workspace/projects
python start_server.py
```

2. **访问网页**
- PC端会自动打开浏览器
- 或手动访问：http://localhost:8080/resume-chat.html

3. **手机访问**
- 确保手机和电脑在同一WiFi下
- 查看电脑IP地址：`ifconfig` (Mac/Linux) 或 `ipconfig` (Windows)
- 手机浏览器输入：`http://你的IP:8080/resume-chat.html`

### 停止服务器
按 `Ctrl + C`

## 🌐 方式二：GitHub Pages部署（永久访问）

### 步骤

1. **创建GitHub仓库**
```bash
# 登录GitHub，创建新仓库：resume-analyzer
```

2. **推送代码**
```bash
cd /workspace/projects
git init
git add assets/resume-chat.html
git commit -m "Add resume analyzer web page"
git branch -M main
git remote add origin https://github.com/你的用户名/resume-analyzer.git
git push -u origin main
```

3. **启用GitHub Pages**
   - 进入仓库Settings → Pages
   - Source选择：Deploy from a branch
   - Branch选择：main，/ (root)
   - 点击Save

4. **访问网页**
```
https://你的用户名.github.io/resume-analyzer/resume-chat.html
```

### 优点
- ✅ 完全免费
- ✅ HTTPS安全访问
- ✅ 全球CDN加速
- ✅ 永久有效

## 📱 方式三：集成到微信公众号

### 配置步骤

1. **登录微信公众平台**
   - 进入：设置与开发 → 公众号设置 → 功能设置
   - 配置：JS接口安全域名（添加你的域名）

2. **添加菜单**
   - 进入：自定义菜单
   - 新建菜单：简历分析
   - 跳转网页：输入GitHub Pages链接

3. **用户访问**
   - 用户点击菜单即可使用

## ❓ 常见问题

**Q: 网页打不开怎么办？**
A: 检查是否启动了本地服务器，或GitHub Pages是否配置成功

**Q: 手机无法访问本地服务器？**
A: 确保手机和电脑在同一WiFi，检查防火墙设置

**Q: 分析失败？**
A: 检查网络连接，确保API服务正常
