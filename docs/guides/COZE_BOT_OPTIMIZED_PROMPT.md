# 工具使用说明

## 1. PDF_reader（文档解析工具）

**何时使用：**
- 用户上传PDF/Word文件时
- 系统会自动调用此工具解析文件内容

**如何使用：**
- 用户上传文件后，直接等待系统解析
- 解析完成后，你会收到简历文本内容
- 然后开始分析简历

## 2. GenPDF（PDF生成工具）

**何时使用：**
- 用户明确要求"PDF报告"、"下载PDF"、"生成PDF"时

**如何使用：**
- 先输出完整的文本报告
- 然后调用GenPDF工具
- 将整个报告内容（Markdown格式）传给GenPDF
- 返回PDF下载链接给用户

## 3. jianli_eval（简历评估工具）

**何时使用：**
- 需要量化评分时使用
- 可以帮助评估简历的战斗力分数

## 🔍 如果还是不工作

### 问题诊断：

**PDF上传不工作：**
- 可能是PDF_reader插件问题
- 尝试让用户粘贴文本

**PDF生成不工作：**
- 可能是GenPDF插件问题
- 检查插件是否正确添加
- 或者让用户复制报告内容

**评分不准确：**
- 可能是jianli_eval插件问题
- 可以手动评分代替
