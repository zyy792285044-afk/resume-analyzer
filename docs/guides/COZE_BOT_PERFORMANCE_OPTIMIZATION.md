# Coze Bot 性能优化指南

## 🐛 问题：回复速度慢

**可能原因：**
1. 系统提示词太长
2. 模型配置不当
3. 分析流程太复杂
4. 没有启用流式输出
5. 网络或平台性能问题

---

## ✅ 优化方案

### 方案一：简化系统提示词（推荐）

**问题：**
- 系统提示词太长会增加处理时间
- Bot需要先理解所有指令才能开始工作

**解决方案：**
使用精简版提示词（见下方）

---

### 方案二：优化模型配置

**推荐配置：**

| 参数 | 推荐值 | 说明 |
|------|-------|------|
| 模型 | 豆包·Seed 1.6 | 更快，适合简单任务 |
| Temperature | 0.3 | 保持稳定输出 |
| 最大输出长度 | 8000 | 减少输出长度 |
| 流式输出 | 开启 | 必须开启！ |

**为什么选择豆包·Seed 1.6：**
- ✅ 速度快（比Seed 2.0快2-3倍）
- ✅ 质量足够好
- ✅ 成本更低

**如果需要更高质量：**
- 使用豆包·Seed 2.0 Pro
- 但速度会慢一些

---

### 方案三：优化工作流程

**问题：**
- 分析流程太复杂（5步分析）
- 每步都要处理，增加时间

**解决方案：**
- 简化分析流程
- 减少不必要的步骤

---

## 📝 精简版系统提示词（快速版）

**直接复制到Coze Bot：**

```
# 角色
你是简历分析专家，帮助用户快速分析简历并给出优化建议。

# 工作流程

1. **接收简历**
   - 用户上传文件或粘贴文本
   - 等待系统解析（如果是文件）

2. **快速分析**
   评估4个维度（各25分，总分100）：
   - 技能匹配度
   - 项目质量
   - 表达清晰度
   - 简历完整性

3. **输出报告**
   使用简洁格式，避免冗长描述

4. **生成PDF**（用户要求时）
   - 调用GenPDF工具
   - 返回下载链接

---
```

## 🔍 其他优化建议

### 1. 检查网络

**如果还是很慢：**
- 尝试切换网络（WiFi/4G/5G）
- 清除浏览器缓存
- 使用其他浏览器测试

### 2. 减少插件使用

**插件会增加处理时间：**
- 只保留必要的插件
- 如果PDF生成慢，可以考虑去掉
- 让用户复制报告内容

### 3. 分段输出

**让Bot先给简短回复：**

在提示词中添加：
```
# 输出策略
1. 先输出战斗力评分（1-2秒）
2. 再输出主要问题（3-5秒）
3. 最后输出优化建议（5-10秒）
```

## 📊 性能基准

### 正常响应时间参考

| 操作 | 预期时间 | 备注 |
|------|---------|------|
| 文件上传解析 | 2-5秒 | 取决于文件大小 |
| 首次文字输出 | 3-8秒 | 开始看到回复 |
| 完整分析报告 | 10-30秒 | 完整输出 |
| PDF生成 | 5-10秒 | 调用GenPDF |

### 异常情况

**如果超过这些时间：**
- 首次响应 > 15秒 → 检查模型配置
- 完整分析 > 60秒 → 简化提示词
- PDF生成 > 20秒 → 可能是插件问题

   - 是否设置了过长的输出？
   - 推荐：8000 tokens

4. ✅ 系统提示词长度
   - 提示词是否太长？
   - 推荐：< 1000字

- 或截图保存

- 只给总分，不给分项
