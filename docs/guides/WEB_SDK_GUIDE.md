# Coze Web-SDK 快速部署指南

## 🎯 最简单的方案：Web-SDK

**优点：**
- ✅ 无需任何编程
- ✅ 无需部署服务器
- ✅ 一键生成访问链接
- ✅ 直接使用Coze后端
- ✅ 支持所有功能（文件上传、PDF解析等）

---

## 📋 具体步骤

### 第一步：获取项目信息

1. 在Coze编程项目页面，点击右上角"部署"
2. 选择"Web-SDK"选项卡
3. 你会看到以下信息：
   - **App ID**: 你的应用ID
   - **API Key**: 你的API密钥
   - **Web-SDK地址**: SDK的CDN地址

### 第二步：生成访问链接

Coze提供了两种方式：

#### 方式一：官方提供的预览链接

在部署页面，Coze会直接提供一个预览链接，格式如：
```
https://applink.coze.cn/你的AppID
```

**这个链接可以：**
- ✅ 直接访问使用
- ✅ 分享给任何人
- ✅ 支持所有功能
- ✅ 无需登录（如果设置了公开访问）

#### 方式二：生成独立网页（推荐）

Coze会提供一个HTML代码片段，你可以：

1. 复制代码
2. 保存为HTML文件
3. 上传到任何静态托管（GitHub Pages、Vercel等）
4. 获得独立的访问链接

---

## 🚀 快速操作指南

### 在Coze编程项目中：

1. **打开你的项目**
2. **点击右上角"部署"按钮**
3. **选择"Web-SDK"标签**
4. **查看两个选项：**

   **选项A：直接访问链接**
   ```
   https://applink.coze.cn/你的AppID
   ```
   - 点击即可使用
   - 可以分享给他人
   
   **选项B：嵌入代码**
   ```html
   <script src="https://cdn.coze.cn/sdk.js"></script>
   <script>
     coze.init({
       app_id: "你的AppID",
       // 其他配置
     });
   </script>
   ```

## 🎨 嵌入示例

如果你想在其他网站使用：

```html
<!DOCTYPE html>
<html>
<head>
    <title>简历战斗力分析</title>
</head>
<body>
    <!-- 粘贴Coze提供的Web-SDK代码 -->
    <div id="coze-chat"></div>
    <script src="https://cdn.coze.cn/sdk.js"></script>
    <script>
        coze.init({
            app_id: "你的AppID",
            container: "#coze-chat",
            theme: "light",
            // 其他配置项
        });
    </script>
</body>
</html>
```

## ⚠️ 注意事项

1. **API Key安全**
   - 不要在前端代码中暴露API Key
   - 使用Web-SDK会自动处理认证

2. **访问权限**
   - 如果要公开访问，需要在部署设置中选择"公开"
   - 如果要限制访问，可以设置访问密码

3. **功能支持**
   - Web-SDK支持所有功能
   - 包括文件上传、PDF解析、流式输出等
