## 插件配置（重要！）

### 必须添加的插件：
1. **文档解析** - 用于解析PDF/Word文件
   - 在插件市场搜索"文档解析"
   - 选择支持"文件上传"的插件
   - 不要使用"URL解析"的插件（会有证书问题）

2. **PDF生成** - 用于生成PDF报告
   - 在插件市场搜索"PDF生成"
   - 添加到Bot

## 使用方式

### 正确方式：
- 用户在对话框点击"上传文件"按钮
- 选择PDF文件上传
- Bot自动解析并分析

### 错误方式（会导致证书问题）：
- 用户提供PDF的URL链接
- Bot尝试从URL下载PDF

## 发布步骤

1. 创建Bot并添加上述提示词
2. 添加"文档解析"插件（支持文件上传的）
3. 添加"PDF生成"插件
4. 测试：上传PDF文件 → 分析 → 生成PDF报告
5. 发布到"Bot商店"（公开）
6. 获得：`https://www.coze.cn/store/bot/你的BotID`

## 获取API密钥（可选）

如果需要API调用：
1. 发布Bot后，点击"API"
2. 创建API密钥
3. 获得：
   - Bot ID
   - API Key
   - API端点
